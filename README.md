# Kaggle_Heart_Analysis

Goal : To predict whether an individual will experience heart attack or not (binary classification) based on a variety of features. 

Using Kaggle's dataset from the following URL : # https://www.kaggle.com/rashikrahmanpritom/heart-attack-analysis-prediction-dataset 
EDA, and machine learning was done using python. Extensive use of scikit library was done to build the model, create testing and training datasets and 
further to evaluate the accuracy of the model. 

Data used has been uploaded. The csv file is named as heart.csv .
